隐私政策

生效日期：本隐私政策自您使用 Bangumi App 之日起生效。

Bangumi App（以下简称"本应用"）非常重视您的隐私保护。本隐私政策旨在向您说明我们如何处理您的个人信息，以及我们如何保护您的隐私权。请您仔细阅读本隐私政策。

1. 信息收集与存储

为了提供更好的使用体验，本应用会在您的设备本地存储以下信息：

  • 登录认证信息：用于保持您的登录状态，使您无需重复登录
  • 条目数据缓存：用于提高应用响应速度，减少网络请求
  • 用户偏好设置：用于保存您的个性化配置

重要说明：上述所有信息仅存储在您的设备本地，我们不会收集、传输或存储任何信息到我们的服务器。您的数据完全由您掌控。

2. 登录信息管理

当您使用本应用进行登录操作时：

  • 本应用使用 bgm.tv 网站提供的开发者平台 API 进行身份验证
  • 您的用户名和密码将在与应用隔离的网页浏览器中直接提交到 bgm.tv 网站
  • 本应用不会接触、存储或传输您的登录密码
  • 为了维持登录状态，本应用会存储由 bgm.tv API 返回的认证令牌（Access Token）
  • 认证信息采用 iOS 系统 Keychain 服务进行加密存储，确保安全性
  • 当您在应用中退出登录时，所有认证信息将从 Keychain 中立即删除

3. 数据安全

我们采用以下措施保护您的数据安全：

  • 使用 iOS Keychain 服务加密存储敏感认证信息
  • 所有数据仅存储在您的设备本地，不会上传到任何服务器
  • 应用与 bgm.tv 服务器的通信均通过 HTTPS 加密传输

4. 信息收集与使用

本应用不会收集、传输或分享任何个人信息。我们不会：

  • 收集您的个人身份信息
  • 追踪您的使用行为
  • 向第三方分享或出售您的信息
  • 使用任何分析工具或广告追踪

5. 第三方服务

本应用通过 bgm.tv 开发者平台 API 获取数据。您的数据使用受 bgm.tv 的隐私政策约束。我们建议您查阅 bgm.tv 的隐私政策以了解更多信息。

6. 数据删除

您可以随时通过以下方式删除本应用存储的所有数据：

  • 在应用内退出登录，这将删除所有认证信息
  • 删除应用缓存数据（通过 iOS 设置 > 通用 > iPhone 存储空间）
  • 卸载并重新安装应用

注意：由于 Keychain 中的认证信息在应用删除后可能不会自动清除，如果您希望彻底清空所有个人信息，建议您先退出登录，然后再删除应用。

7. 儿童隐私

本应用不面向 13 岁以下的儿童。我们不会故意收集儿童的个人信息。

8. 隐私政策更新

我们可能会不定期更新本隐私政策。重大变更时，我们会在应用内或通过其他方式通知您。继续使用本应用即表示您接受更新后的隐私政策。

9. 联系我们

如果您对本隐私政策有任何疑问或建议，请通过应用内的反馈渠道联系我们。
